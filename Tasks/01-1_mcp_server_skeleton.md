# Task 01-1 - MCP 서버 기본 골격과 연결 확인

## 설명
Node.js와 `@modelcontextprotocol/sdk`를 사용해 MCP 서버를 실행할 수 있어야 하며, MCP inspector 같은 클라이언트가 연결했을 때 서버 이름과 tool 목록을 확인할 수 있어야 한다. 실제 생성 기능은 아직 포함하지 않고, 서버가 정상적으로 떠 있는지 확인할 수 있는 최소한의 tool 1개만 제공하는 단계를 이번 task의 목표로 한다.

## 구현 항목
- [x] MCP 서버를 실행할 수 있는 기본 골격이 준비된다.
- [x] 클라이언트가 연결했을 때 서버 이름이 정상적으로 노출된다.
- [x] 서버 기동 확인용 최소 tool 1개가 목록에 노출되고 호출 가능하다.
- [x] MCP inspector에서 연결과 tool 목록 확인이 가능하다.

## 범위 밖
- 실제 스크립트 생성, 이미지 생성, 영상 생성 tool 구현
- 외부 AI API 연동
- 여러 개의 보조 tool 추가
